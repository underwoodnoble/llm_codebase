# Large Language Model Codebase
本仓库基于Huggingface集成了一系列大语言模型训练方法，旨在帮助初学者更快的入门大语言模型的训练。

## Overview
* 本仓库目前支持的任务有：Classification, Reward Model Training, Supervised Finetuning, Rejection Sampling

* 本仓库目前支持的模型有：bert, llama, alpaca
